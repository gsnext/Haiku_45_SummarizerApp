About GenAIsummarizer

GenAIsummarizer is a self-hosted, AI-powered application designed to help users quickly understand document content by generating intelligent, concise summaries. Built entirely in Python, it offers both a user-friendly web interface and a powerful REST API for seamless integration.

Key Features

Multi-Format Support

Summarize text, PDF, DOCX files, and web URLs all in one place

Configurable Length

Choose from Short, Medium, or Long summaries based on your needs

REST API

Integrate summarization into your applications with our comprehensive API

Batch Processing

Process up to 10 files in a single batch request for efficiency

Summary History

Maintain a searchable history of all your previous summaries

Secure Authentication

JWT token-based authentication for API endpoints

Technical Stack

  • Backend: Python with FastAPI framework
  • Frontend: HTML, CSS, JavaScript with Jinja2 templating
  • AI Model: Azure OpenAI for summarization
  • Document Processing: Support for PDF, DOCX, and web content
  • Authentication: JWT tokens via python-jose

Getting Started

  1. Create and activate a Python virtual environment
  2. Install dependencies using pip install -r requirements.txt
  3. Set up environment variables for Azure OpenAI credentials
  4. Run the application using python run.py
  5. Access the web UI at http://localhost:8000

API Endpoints

The following REST API endpoints are available:

  • POST /api/summarize - Summarize plain text
  • POST /api/summarize/file - Summarize uploaded files
  • POST /api/summarize/url - Summarize web content
  • POST /api/batch - Batch process multiple items
  • GET /api/history - Retrieve user's summary history
  • GET /api/summary/{id} - Get a specific summary
  • DELETE /api/summary/{id} - Delete a summary

Security & Configuration

GenAIsummarizer follows security best practices:

  • Secrets and API keys are managed through environment variables
  • JWT tokens are used for API authentication
  • File uploads are validated for format and size
  • All actions and errors are logged for audit purposes
  • CORS is configured for secure cross-origin requests

Requirements

  • Python 3.8 or higher
  • Azure OpenAI subscription and credentials
  • Minimum system: 2GB RAM, 100MB disk space
  • Compatible with Windows, Linux, and macOS
